### Estrutura do Repositório

Cada pasta representa um tema específico do PHP. Dentro de cada pasta há arquivos com exercícios relacionados ao assunto estudado.

---
### Como Executar os Exercícios

Os exercícios são executados em um **servidor local** usando o **XAMPP**.

#### Como rodar:

1. Instalar o **XAMPP**
2. Iniciar o módulo **Apache**
3. Clonar este projeto na pasta: htdocs
4. Acessar no navegador:
http://localhost/nome-da-pasta-do-projeto
5. Abrir **no navegador** a pasta do exercício que você deseja executar.

> PHP é uma linguagem que roda no **servidor**, por isso os arquivos não funcionam corretamente apenas abrindo no navegador sem o XAMPP.
